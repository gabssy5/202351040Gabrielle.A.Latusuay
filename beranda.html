sekilas tentang saya

Perkenalkan, nama saya Gabrielle A. Latusuay. Saya lahir di Ambon, pada tanggal 2 Juni 2005, dan saat ini saya berusia 19 tahun. Saya tinggal di Passo, Lembah Anggrek, dan merupakan mahasiswa aktif semester empat pada Program Studi Ilmu Komputer, Fakultas Matematika dan Ilmu Pengetahuan Alam, Universitas Pattimura. Riwayat pendidikan saya dimulai dari SD Negeri 3 Tomalima Passo, kemudian melanjutkan ke SMP Pengharapan, dan setelah itu menempuh pendidikan di SMA Pengharapan. Setelah menyelesaikan pendidikan menengah, saya melanjutkan studi di Universitas Pattimura, tempat saya belajar hingga saat ini. Di luar kegiatan akademik, saya memiliki beberapa minat dan hobi. Saya menyukai musik dan juga game digital, yang sering saya jadikan sebagai sarana hiburan dan relaksasi di waktu luang. Kedua hal tersebut membantu saya untuk menjaga keseimbangan antara kegiatan belajar dan istirahat. Selain itu, saya juga memiliki ketertarikan dalam bidang pemrograman. Saat ini saya sedang mempelajari dua bahasa pemrograman, yaitu C++ dan Python. Meskipun saya masih dalam tahap belajar, saya berusaha memahami konsep dasarnya dengan tekun dan konsisten. Demikian perkenalan diri dari saya. Terima kasih atas perhatian dan kesempatannya.
